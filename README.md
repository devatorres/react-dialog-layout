## 📋 Descripción

Proyecto simple donde se desarrolla un Dialog atractivo y moderno que se adapta a todas las pantallas.

## 🔨 Cómo arrancarlo

Hay una posibilidad de ver una preview desplegada en _Netlify_ [https://creative-smakager-190ad7.netlify.app/](https://creative-smakager-190ad7.netlify.app/). Pero como programador seguro que llama más la atención usar un poco de npm 😉. Es tan sencillo como clonar el proyecto `git clone <url>`, instalar las dependencias `npm install` y ejecutarlo `npm run dev`.
